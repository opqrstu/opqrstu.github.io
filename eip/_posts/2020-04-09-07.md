>- 재귀 함수도 나올 수 있음

C언어는 함수 지향 언어로 처음 시작할 때 입력하는 main()도 함수다. 함수에는 C언어가 내장되어 있는 내장 함수와 사용자가 만들어 사용하는 사용자 함수가 있다.

## 사용자 정의 함수

사용자 정의 함수는 사용자가 필요한 기능을 취향대로 만들어 사용할 수 있는 함수이다.

사용자 정의 함수를 사용하면 프로그램 구조가 간단해지고 이해하기 쉬워지고 반복입력하는 수고를 줄일 수 있다.

```c
func(i,j)
int i,j;
{
    i*= 3;
    j/= 3;
    printf("%d, %d\n", i, j);
    return i;
}
```
이런 식으로도 정의가능
