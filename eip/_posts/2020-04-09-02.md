## 관계 연산자

관계연산자는 두수를 비교해서 참 또는 거짓 결과를 얻는거    
거짓은 0 참은 1이지만 0이외의 모든 수는 참

== != > >= < <= 왼쪽이 기준이라 `> 는 크다` 라고 함

## 비트 연산자

비트연산자는 비트별로 0, 1 계산해서 결과를 얻는거

|연산자|의미|비고|
|---|---|---|
|&|and|둘다 1이면 1|
|^|xor|같으면 0 다르면 1|
|&#124;|or|하나라도 1이면 1|
|&#126;|not|0이면 1, 1이면 0|
|<<|왼쪽시프트|왼쪽으로 이동|
|>>|오른쪽 시프트|오른쪽이동|

**음수일때 >> 하면 왼쪽에 1이 채워진다함 첨앎**

## 논리 연산자

두 개의 논리 값을 연산해서 참인지 거짓인지 얻는거 거짓 0 참은 1
